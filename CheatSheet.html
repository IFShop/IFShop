CheatSheet - Commands for Windows & Linux

Windows - File Management

del Show example
Deletes a file at the specified path. Example: del C:\\path\\to\\file.txt
move Show example
Moves a file from one location to another. Example: move C:\\path\\to\\file.txt D:\\destination
copy Show example
Copies a file from one location to another. Example: copy C:\\path\\to\\file.txt D:\\destination
mkdir Show example
Creates a new directory (folder). Example: mkdir NewFolder
rmdir Show example
Removes a directory at the specified path. Example: rmdir C:\\path\\to\\folder

Linux - File Management

rm Show example
Removes a file at the specified path. Example: rm /path/to/file
mv Show example
Moves a file from one location to another. Example: mv /path/to/file /new/location
cp Show example
Copies a file from one location to another. Example: cp /path/to/file /new/location
mkdir Show example
Creates a new directory (folder). Example: mkdir NewFolder
rmdir Show example
Removes a directory at the specified path. Example: rmdir /path/to/folder

Windows - Networking

ipconfig Show example
Displays network configuration information. Example: ipconfig /all
ping Show example
Tests connectivity to a network host. Example: ping www.google.com
netstat Show example
Displays active network connections. Example: netstat -an

Linux - Networking

ifconfig Show example
Displays network interface configuration. Example: ifconfig
ping Show example
Tests connectivity to a network host. Example: ping www.google.com
netstat Show example
Displays active network connections. Example: netstat -an

Windows - System Information

systeminfo Show example
Displays detailed system information. Example: systeminfo
tasklist Show example
Lists all running tasks. Example: tasklist
taskkill Show example
Kills a running task. Example: taskkill /F /PID 1234

Linux - System Information

uname -a Show example
Displays system information. Example: uname -a
top Show example
Displays real-time system processes. Example: top
kill Show example
Terminates a process by PID. Example: kill -9 1234
